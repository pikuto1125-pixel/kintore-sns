## 【即時実行】初回セットアップ（1回だけ実行）


## 【スケジュールタスク1】毎朝 7:30 — 業務Tips投稿

```
トリガー：毎日 07:30 JST

実行内容：
1. 次のファイルを読む：
   C:\Users\sota4\Desktop\CompanyHQ\Reports\x_posts_week1.md

2. 【知識・Tips系】セクション（#1〜#8）、【Before/After 事例系】（#13〜#15）、
   【価値観・哲学系】（#16〜#20）のうち、まだ「✅投稿済」マークが付いていない
   最も若い番号のネタを1本選ぶ。

3. そのネタの本文をX（pikutoアカウント）に投稿。
   ※ハッシュタグは付けない（先頭2週間は純粋な情報発信優先）

4. 投稿後、x_posts_week1.md 内の該当ネタの直上に
   「✅投稿済 YYYY-MM-DD 朝」を追記して上書き保存。

5. 未投稿ストックが朝+夜合計で3本以下になったら、
   C:\Users\sota4\Desktop\CompanyHQ\Tasks\current_tasks.md
   に「[DX] Week2投稿ネタ生成が必要 @due:明日」を追加。
```
